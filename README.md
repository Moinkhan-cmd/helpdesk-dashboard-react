# Helpdesk Dashboard

A simple helpdesk ticket management UI built with React and Vite.

## Features

- Ticket list view
- Ticket details panel
- Sidebar navigation
- Search functionality

## How to Run

1. Install dependencies:
```
npm install
```

2. Start development server:
```
npm run dev
```

3. Open http://localhost:5173 in browser

## Tech Stack

- React
- Vite
- Lucide React Icons
